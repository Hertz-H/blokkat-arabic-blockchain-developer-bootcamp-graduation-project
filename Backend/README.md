## 🏗️ Project Structure (Foundry)

```
Backend/
├── lib/                      # Foundry dependencies
├── src/                      # Main contracts
│   ├── BookStore.sol         # Core store logic
├── test/                     # Foundry tests
│   ├── BookStore.t.sol       # Main test suite
├── script/                   # Deployment scripts
├── foundry.toml              # Configuration
└── README.md                 # This file
```

## 🚀 Quick Start

### Prerequisites

- [Foundry](https://getfoundry.sh)

### Installation

```bash
cd Backend
forge install
```

## 🧪 Testing

```bash
cd Backend
forge test
```
